# Actions-OpenWrt
Build OpenWrt using GitHub Actions | 使用 GitHub Actions 云编译 OpenWrt带有 闪讯拨号插件 Netkeeper的OpenWrt编译项目
默认适用于软路由x86 64
## Tips
SSH 连接到 Actions
- 复制 SSH 连接命令粘贴到终端内执行，或者复制链接在浏览器中打开使用网页终端。（网页终端可能会遇到黑屏的情况，按 Ctrl+C 即可）
- cd openwrt && make menuconfig
- 完成后按Ctrl+D组合键或执行exit命令退出，后续编译工作将自动进行。
